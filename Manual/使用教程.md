# XCMOD 修改器使用教程

## 一、软件简介

XCMOD 是一款功能强大的游戏内存修改工具，支持内存扫描、数值修改、插件管理等功能，帮助玩家轻松修改游戏数据。

## 二、安装与启动

### 2.1 安装要求
- 操作系统：Windows 10/11
- 无需安装，直接运行即可

### 2.2 启动方式
1. 下载并解压 XCMOD 压缩包
2. 双击 `XCMOD.exe` 启动程序
3. 首次启动会自动创建 `games/` 目录用于存放插件配置

## 三、基本使用流程

### 3.1 选择目标进程

1. 启动游戏
2. 在 XCMOD 左侧"进程列表"中找到游戏进程
3. 点击进程名称选中（绿色高亮表示已选中）

### 3.2 内存扫描（查找地址）

1. 在"内存扫描"面板输入游戏中看到的数值（如生命值、金币等）
2. 点击【首次扫描】
3. 返回游戏，改变该数值（如消耗生命值、花费金币）
4. 在修改器中输入新的数值，点击【筛选扫描】
5. 重复步骤 3-4，直到找到唯一的内存地址

### 3.3 添加修改项

1. 在扫描结果中右键点击找到的地址
2. 选择【添加到当前游戏】
3. 输入修改项名称（最多10个汉字）
4. 设置目标值和数据类型
5. 点击【保存】

## 四、插件管理

### 4.1 上传插件

1. 点击右侧【上传插件】按钮
2. 选择以下方式之一：
   - 点击【选择文件】上传 JSON 插件文件
   - 点击【粘贴 JSON】直接粘贴插件配置
   - 点击【下载 JSON-Demo】获取示例配置

### 4.2 创建插件配置

插件是 JSON 格式的配置文件，示例如下：

```json
{
  "id": "example-game",
  "name": "示例游戏",
  "executable": "Game.exe",
  "description": "这是一个示例插件",
  "cheats": [
    {
      "id": "unlimited-health",
      "name": "无限生命",
      "address": "0x12345678",
      "value": 999,
      "dataType": "int"
    },
    {
      "id": "infinite-money",
      "name": "无限金钱",
      "address": "0x1234567C",
      "value": 999999,
      "dataType": "int"
    }
  ]
}
```

### 4.3 删除修改项

1. 鼠标悬停在修改项卡片上
2. 点击右上角的【×】按钮
3. 确认删除即可

## 五、修改项操作

### 5.1 单次修改

1. 在修改项输入框中输入目标值
2. 点击【单次修改】按钮
3. 游戏中的数值会立即更新

### 5.2 锁死（循环修改）

1. 在修改项输入框中输入目标值
2. 点击【锁死】按钮（变为绿色表示已激活）
3. 数值会被持续锁定，不会被游戏修改
4. 点击【取消】停止锁定

## 六、数据类型说明

| 类型 | 说明 | 范围 |
|------|------|------|
| int | 整数 | -2147483648 ~ 2147483647 |
| float | 单精度浮点 | 约 ±3.4×10³⁸ |
| double | 双精度浮点 | 约 ±1.7×10³⁰⁸ |
| byte | 字节 | 0 ~ 255 |
| short | 短整数 | -32768 ~ 32767 |

## 七、常见问题

### Q1：找不到游戏进程？

- 确保游戏已启动
- 尝试点击【刷新进程列表】
- 检查游戏是否以管理员身份运行

### Q2：扫描不到地址？

- 确保输入的数值与游戏中一致
- 尝试切换不同的数据类型
- 某些游戏使用动态地址，需要使用指针

### Q3：修改后数值会自动恢复？

- 使用【锁死】功能持续锁定数值
- 检查是否使用了正确的数据类型

### Q4：插件上传失败？

- 检查 JSON 格式是否正确
- 确保所有必填字段都已填写
- 注意地址必须以 `0x` 开头

## 八、安全提示

1. 修改游戏内存可能违反游戏用户协议
2. 在线游戏使用修改器可能导致账号封禁
3. 使用前请备份游戏存档
4. 仅在单机游戏或私人服务器中使用
