    📝 插件制作教程

    1. 创建插件配置

    插件是 JSON 格式的配置文件，放置在 games/ 目录下：

    {
  "id": "example-game",
  "name": "示例游戏",
  "executable": "Game.exe",
  "description": "这是一个示例插件配置",
  "cheats": [
    {
      "id": "unlimited-health",
      "name": "无限生命",
      "description": "设置生命值为999",
      "address": "0x12345678",
      "value": 999,
      "dataType": "int"
    },
    {
      "id": "infinite-money",
      "name": "无限金钱",
      "description": "设置金钱为999999",
      "address": "0x1234567C",
      "value": 999999,
      "dataType": "int"
    },
    {
      "id": "god-mode",
      "name": "无敌模式",
      "description": "开启无敌状态",
      "address": "0x12345680",
      "value": 1,
      "dataType": "byte"
    }
  ]
}

    2. 数据类型说明

    类型适用场景
    int生命值、金钱、等级等整数
    float百分比、坐标等小数
    byte开关状态（0/1）
    short小范围整数
    double高精度小数

    3. 上传规范

    • 文件名使用英文小写，用连字符分隔
    • 游戏名称使用中文全称
    • 内存地址必须以 0x 开头
    • 每个修改项必须有唯一的 id
